# To-Do List 

A simple to-do list application built with Python and Tkinter. Users can add tasks and mark them as completed.

## Description

This application allows users to:
- Add new tasks to a list
- Mark tasks as completed using checkboxes
- Remove completed tasks from the list

No database is used - all data is stored in memory during execution.

## Technologies Used

- Python 3
- Tkinter (GUI library)
